Overview

This project is a collection of basic HTML and CSS web pages created to practice fundamental web development concepts such as webpage structure, lists, hyperlinks, images, text formatting, and styling using external CSS. The project demonstrates how different HTML elements and CSS properties work together to create simple web pages.

Topics Covered
HTML Structure

HTML structure defines the basic layout of a webpage using elements like headings, paragraphs, body, and containers.

Lists

Lists are used to organize content in ordered or unordered format.

Links

Links are created using anchor tags to connect webpages or external websites.

Images

Images are added to webpages using the <img> tag for visual representation.

Text Formatting

HTML formatting tags are used to style text such as bold, italic, underline, and headings.

External CSS Styling

External CSS is used to apply styles separately from HTML for better design and maintainability.
